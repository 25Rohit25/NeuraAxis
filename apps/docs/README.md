# @neuraxis/docs

> Documentation Site for NEURAXIS Platform

## Overview

This package contains the documentation website for NEURAXIS. It includes API documentation, user guides, and developer resources.

## Structure

```
docs/
├── api/              # API reference documentation
├── guides/           # User and developer guides
├── architecture/     # System architecture docs
└── tutorials/        # Step-by-step tutorials
```

## Building

```bash
npm run build
```

## Development

```bash
npm run dev
```
